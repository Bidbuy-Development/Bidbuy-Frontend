BIDBUY FRONTEND - TEAM SETUP GUIDE

QUICK START - GET UP AND RUNNING IN 5 MINUTES

1. CLONE AND SETUP
Open your terminal and run these commands:

cd Bidbuy-Frontend
npm install
npm run dev

Open http://localhost:3000 in your browser and you're ready to go!

COLOR SYSTEM - USE THESE EVERYWHERE

In globals.css we have defined CSS variables for consistent colors:

Primary Colors:
--color-primary-purple: #9267a0    ← Main brand color
--color-primary-gold: #b87a08      ← Secondary brand color

Text Colors:
--color-text-title: #333333        ← Main text color
--color-secondary-text: #999999    ← Lighter text color

Background Colors:
--color-background-card: #f2f2f2   ← Card backgrounds
--color-background-color: #fcfcfc  ← Main page background
--color-input-background: #d9d9d95c ← Input field backgrounds

Accent Colors:
--color-accent-yellow: #f4b400     ← Accent/highlight color
--color-accent-light-yellow: #fff9e5 ← Light accent

HOW TO USE COLORS IN YOUR CODE:

Instead of hardcoding colors, use the predefined Tailwind color classes:

CORRECT (Tailwind color classes):
bg-primary-purple
text-primary-gold
bg-background-card
text-text-title
text-secondary-text
bg-accent-yellow

WRONG:
background-color: #9267a0;  ← Don't hardcode colors!
bg-[#9267a0]  ← Don't use hex codes directly!

UI COMPONENTS - READY TO USE

Button Component
Location: /src/components/UI/Button.js

Usage Examples:
import Button from "../UI/Button";

// Primary button
<Button variant="primary" radius="full">Click Me</Button>

// Secondary button
<Button variant="secondary" radius="lg">Secondary</Button>

// With icon from react icons
<Button variant="primary" hasIcon={<IconName />}>Button with Icon</Button>

Button Properties:
- variant: "primary" or "secondary"
- radius: "none", "sm", "md", "lg", "xl", or "full"
- hasIcon: Import icon from react icons and use like the example above
- onClick: click handler function

Input Component
Location: /src/components/UI/Input.js

Usage Examples:
import Input from "../UI/Input";

<Input 
  label="Email Address" 
  placeholder="Enter your email"
  inputStyle="w-full"
/>

Toast Notifications (react-toastify)
Already set up and ready to use! Show success, error, or info messages to users.

Usage Examples:
import { toast } from "react-toastify";

// Success toast
toast.success("✅ Success message!");

// Error toast
toast.error("❌ Error message!");

// Info toast
toast.info("ℹ️ Info message!");

// Custom toast
toast("🦄 Custom message!");

// In a button click
<Button onClick={() => toast.success("Button clicked!")}>
  Click Me
</Button>

TROUBLESHOOTING

If colors don't work:
- Make sure you're using the color variables: --color-primary-purple
- Check that globals.css is imported in your layout

If components break:
- You don't need to touch the src/app/page.js - All components are imported there already
- Make sure you're importing components correctly
- Check that all props are being passed correctly
